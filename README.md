# 🛡️ Code Guardian: The Anti-Copy-Paste Extension

**"Stop Mindless Coding. Start Learning."**

Code Guardian is a VS Code extension built to break the "AI Copy-Paste" habit. While LLMs make coding faster, they often lead to a lack of deep understanding. Code Guardian forces a "friction point" that ensures developers actually comprehend the code they are adding to their projects.

---

## 🚀 The MVP Workflow
1. **Highlight:** The user highlights a block of code (e.g., from an AI chat).
2. **Explain:** Code Guardian intercepts the action and prompts the user to explain the logic in their own words.
3. **AI Grading:** The explanation is sent to our **AWS Lambda** backend, which uses **Google Gemini 2.5 Flash** to grade the response (0-100%).
4. **The 70% Gate:** If the user scores above 70%, the code is verified.
5. **ELI5 Fallback:** If the user fails 3 times, the extension triggers an **"Explain Like I'm 5" mode**, using simple analogies (generated by Gemini) to teach the core concept before the user moves on.

---

## 🛠️ Technical Architecture

[Image of a sequence diagram showing VS Code Extension calling AWS API Gateway to Lambda to Gemini API]

* **Frontend:** VS Code Extension API (TypeScript)
* **Backend:** AWS Lambda (Python 3.12) - Serverless Architecture
* **AI Engine:** Google Gemini 2.5 Flash API
* **Database:** Amazon DynamoDB (Session tracking, scoring history, and strike counting)
* **API Layer:** AWS API Gateway (REST Endpoint)
